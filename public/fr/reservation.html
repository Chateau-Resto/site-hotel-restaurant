Informations Générales

Les chambres sont disponibles à partir de 15h00.
Nous vous remercions de libérer votre chambre avant 11h00.
Le restaurant La Closerie vous accueille tous les jours sauf le dimanche soir, le lundi et le samedi midi.
Pour votre bien être et celui des autres, l’hôtel est entièrement non fumeur.

Un parking est disponible sur le domaine accessible librement entre 8h00 et 22h00 (15h00 le dimanche). Si vous prévoyez une arrivée tardive, merci de contacter la reception de 9h00 à 17h00 pour le code d’accès du portail.
